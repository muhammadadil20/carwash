# Carwash Monorepo

This repository hosts the Carwash platform as a multi-folder workspace.

## Structure
- `carwash-infra/`: Infrastructure as code, CI/CD, and ops assets
- `carwash-microservices/`: Backend services and APIs
- `carwash-mobile/`: Mobile app source (e.g., React Native/Flutter)
- `carwash-web/`: Web frontend application

More documentation will be added as components evolve.
